## Summary

A 3-step how-to for using [[claude-code]] to redesign a room from a single photo
via the "Banana Skill" + Google AI Studio. Transcript-backed, so the actual steps
are captured (the caption only hinted at it).

## How-To (from transcript)

1. Google "Banana Skill Claude", open the first link, copy its URL. In Claude
   Code, paste the link and say **"install this skill"**.
2. In **Google AI Studio** → API keys → create a new key. Paste it into Claude
   Code and say **"set this up"**.
3. Upload a photo of the room, type **`/Banana`**, and prompt "redesign this room
   like a professional interior designer." Iterate on furniture / colors / layout.

## Action Items

- If a room makeover comes up: follow the 3 steps above (need a Google AI Studio
  API key).
- Reusable pattern worth noting: a Claude Code **Skill** installed from a URL +
  an image-gen backend (Google AI Studio) → repeatable creative workflow. Same
  "install a skill" mechanic we use for the wiki lanes.
