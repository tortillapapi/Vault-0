# AI Botting Guide & Assistant (NotebookLM)

## Overview
Leverage NotebookLM and existing botting documentation to build an AI-powered reference guide and assistant bot for use with retail botting tools — primarily Refract and Stellar. This creates a queryable knowledge base that speeds up drop-day setup, reduces configuration errors, and gives a competitive edge on complex releases.

**Categories:** RARE FORCE ONE + Automation

---

## Goals
- Ingest all relevant botting docs (Refract, Stellar, AIO Bot, etc.) into NotebookLM
- Build a queryable AI guide for drop-day settings, proxies, and task configs
- Reduce time-to-ready on drop days
- Explore automation hooks (via OpenClaw) to push settings/configs from guide → live tool
- Create a personal "botting brain" that improves over time
